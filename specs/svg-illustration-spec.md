# Спецификация SVG-иллюстраций катехизиса (для AI-агента)

Требования к генерации иллюстраций к 114 вопросам детского катехизиса.

**Целевая аудитория:** дети 6–8 лет из консервативных баптистских семей. Отсюда два сквозных приоритета: (1) простота и дружелюбность для маленького ребёнка, (2) строгая доктринальная и нравственная уместность.

### Хранение и сборка

- **Файл (SVG, предпочтительно):** `public/illustrations/qNNN.svg` (`q001.svg` … `q114.svg`).
- **Альтернатива — растр:** `public/illustrations/qNNN.png` \| `.jpg` \| `.jpeg` \| `.webp`. Доктринальные правила §2 / §7 те же; технические правила SVG (§3–§4, палитра, viewBox) к растру не применяются. Рекомендуемый кадр 4:3 (1200×900).
- **JSON:** в `catechism.json` → `questions[].illustration` — только **путь** относительно `public/` (напр. `"illustrations/q001.svg"` или `"illustrations/q001.png"`), не содержимое файла. `null` — путь ещё не задан.
- **Рендер:** `resolveIllustration(n)` из `images/illustrations.node.ts` — SVG инлайнится, растр отдаётся как публичный URL для `<img>`. Если файла нет или `illustration === null` — нейтральный SVG-плейсхолдер (`viewBox="0 0 1200 900"`, цвет `#BFE3F0`).
- **Санитизация:** `sanitizeSvg` снимает XSS-векторы у инлайн-SVG. Это **не** замена чек-листа §9.

## 1. Разбор образцов

Проанализирована иллюстративная часть карточек (сам рисунок, не текстовый блок).

**Что перенимаем:**
- Единый спокойный, тёплый колорит; мягкие формы, скруглённые силуэты.
- Одна ясная смысловая сцена, минимум деталей, много «воздуха».
- Фон на всю область (небо/холмы), без пустоты и без рамок внутри рисунка.
- Дружелюбные, спокойные лица без карикатурности и без страха.
- Природа и творение как основной визуальный язык (образцы 1, 3).

**Что НЕ перенимаем (антипаттерны в образцах):**
- **Образец 4** изображает Христа, **образец 5** — Святого Духа. Изображение Лиц Троицы противоречит 2-й заповеди (вопросы 69–70 катехизиса) и убеждениям аудитории. Не воспроизводить.
- **Образец 2**: женщина в брюках — нарушение требования модести. Не воспроизводить.
- Живописная растровая фактура и свечения (образцы 2, 4, 5) — недостижимы в чистом SVG. Целевой стиль — плоский вектор (образцы 1, 3).

## 2. Жёсткие ограничения (безопасность и доктрина) — приоритет над эстетикой

Эти правила нельзя нарушать ни при каких обстоятельствах. При конфликте с «красотой сцены» — выбирается правило.

1. **Не изображать Бога Отца, Иисуса Христа (лицо/фигуру), Святого Духа.** Никаких антропоморфных фигур Божества, нимбов над «божественными» персонажами, «руки Бога» из облака. Божье действие показывать косвенно: свет/лучи, сцена творения, крест, пустая гробница, раскрытая Библия, корона (символ Царя), хлеб и чаша (символы). Ангелы допустимы как отдельные существа, но без отождествления с Богом.
2. **Модесть — женщины и девочки:** только платья или юбки ниже колена; рукав длинный или 3/4; закрытый вырез; силуэт свободный, не облегающий. **Никаких брюк, шорт, коротких/облегающих или открытых нарядов.** Волосы опрятные, скромные; допустим платок.
3. **Модесть — мужчины и мальчики:** закрытая одежда, длинные рукава или туники/хитоны; без обнажённого торса.
4. **Адам и Ева:** всегда изображать скромно одетыми (простые туники) либо на отдалении/со спины среди листвы сада. **Никакой наготы, никаких намёков.** Сцены «сотворения Евы из ребра», «грехопадения» решать символически и целомудренно (сад, дерево, плод, змей на отдалении).
5. **Без страха для ребёнка 6–8 лет.** Ад, смерть, суд, гнев — только мягко и символически: без огня-мучений с людьми, без тел, без крови, без чудовищ, без пугающих лиц. См. таблицу §7.
6. **Без идолопоклоннических объектов как позитива.** Статуи/идолы допустимы только в назидательном контексте 2-й заповеди (как то, чего делать нельзя), но не как украшение.
7. **Историко-культурная уместность** библейских сцен (одежды-туники, сандалии, ближневосточный пейзаж), без анахронизмов.

## 3. Визуальный стиль

- **Техника:** плоский вектор. Сплошные заливки + простые линейные/радиальные градиенты для неба и мягких свечений. Скруглённые формы, скруглённые концы линий (`stroke-linecap="round"`, `stroke-linejoin="round"`).
- **Запрещено:** имитация живописи, шум/зерно, тяжёлые `<filter>`-блюры, реалистичные тени. Мягкое свечение делать радиальным градиентом, а не размытием.
- **Композиция:** один центральный субъект или простая сцена «передний план + холмы + небо». Правило третей, крупный понятный силуэт, читаемый как иконка на маленьком экране.
- **Настроение:** тёплое, доброе, светлое. Даже серьёзные темы — спокойно.
- **Без текста внутри SVG.** Никаких `<text>` с надписями и никаких букв в рисунке (текст берётся из JSON; ИИ к тому же портит кириллицу). Исключение — `<title>`/`<desc>` для доступности (не отображаются визуально).

### Палитра (фиксированная — источник единообразия всех 114 карточек)

Использовать только эти цвета (или их прозрачные варианты). Не вводить произвольные оттенки.

### Формат холста

- `viewBox="0 0 1200 900"` (4:3), `preserveAspectRatio="xMidYMid meet"`.
- Фон заливается на всю область (без прозрачного фона): небо/пейзаж-подложка.
- Единый viewBox у всех 114 файлов — обязательное условие консистентной вёрстки.

## 4. Технические правила вывода SVG

1. Один корневой `<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 1200 900" preserveAspectRatio="xMidYMid meet">`, самодостаточный.
2. **Без внешних ресурсов:** никаких `<image>` с растром/base64, никаких внешних шрифтов/URL, никаких `<script>`, никаких обработчиков событий.
3. Только векторные примитивы: `path`, `rect`, `circle`, `ellipse`, `polygon`, `polyline`, `line`, `g`, `defs`, `linearGradient`, `radialGradient`, `title`, `desc`.
4. Разумная сложность: ориентир — до ~150 узлов, размер файла до ~15 КБ. Никаких гигантских путей.
5. `<title>` — короткое описание сцены (для скринридеров), напр. «Мальчик на фоне неба». Опционально `<desc>`.
6. Валидный, самозакрывающийся, корректно вложенный XML. Кодировка UTF-8.
7. Цвета — из палитры §3. Допустимо задать их через `<style>`-переменные внутри SVG для будущего ретематизирования.
8. Сохранить в `public/illustrations/qNNN.svg`; убедиться, что `questions[].illustration` = `"illustrations/qNNN.svg"`.

## 5. Кит персонажей (для узнаваемости между карточками)

Повторяющиеся образы должны выглядеть одинаково во всех выпусках.

- **Мальчик-рассказчик** (для вопросов от первого лица: «Кто тебя сотворил?», «А у тебя есть душа?»): короткие каштановые волосы, светлая кожа `#F1C9A5`, жёлтая кофта с длинным рукавом, спокойная добрая улыбка.
- **Девочка** (для парных сцен): скромное платье кораллового или зелёного тона ниже колена, длинный рукав, аккуратные волосы.
- **Библейские фигуры:** простые туники/хитоны, пояс, сандалии; женщины — длинное платье и по возможности платок. Всегда модесть по §2.
- **Природа:** дерево с округлой кроной (`#5E9E5A`) и стволом (`#8B5A2B`); холмы слоями (зелёный → золото); небо-градиент (`#BFE3F0` → `#8FCFE6`); округлые облака.

## 6. Выбор сцены под конкретный вопрос

Ребёнку нужна **конкретная, буквальная, спокойная** сцена, выводимая из ответа, а не абстрактная теология.

Приоритет визуальных решений (от самого предпочтительного):
1. **Творение и природа** — небо, холмы, деревья, животные, вода, солнце/луна/звёзды.
2. **Ребёнок в действии** — молится, читает Библию, слушает, помогает ближнему.
3. **Библейские предметы-символы** — раскрытая Библия, свиток и перо, каменные скрижали, крест, хлеб и чаша, вода крещения (сосуд/река), корона, овцы, путь/дорога, ворота/дверь, свет.
4. **Библейская сцена** (модесто, без Божества) — сад Едемский, ковчег, гора Синай со скрижалями, пустая гробница.

Общее правило: если сцену нельзя показать без изображения Бога — заменить символом (свет, крест, творение, действие человека).

## 8. Промпт для AI-агента (шаблон)

Вставлять на каждый вопрос, подставляя поля. Держать §2 и §3 как неизменяемую «шапку».

```
Ты генерируешь ОДНУ SVG-иллюстрацию для карточки детского катехизиса.
Аудитория: дети 6–8 лет из консервативных баптистских семей.

ВЕРНИ ТОЛЬКО валидный SVG-код, без пояснений и без markdown-обёртки.

ЖЁСТКИЕ ПРАВИЛА (нарушать нельзя):
- НЕ изображай Бога, Иисуса Христа, Святого Духа (ни лицо, ни фигуру, ни руку из облака).
  Божье действие показывай косвенно: свет, творение, крест, пустая гробница, корона, действие человека.
- Модесть: женщины/девочки — только платье или юбка ниже колена, длинный рукав, закрытый вырез,
  свободный силуэт, БЕЗ брюк/шорт/облегающего/открытого. Мужчины/мальчики — закрытая одежда,
  без обнажённого торса. Адам и Ева — только скромно одеты или на отдалении, БЕЗ наготы.
- Без страха: ад/смерть/суд — мягко и символически, без огня с людьми, тел, крови, чудовищ.
- Без текста и букв внутри рисунка (только <title> для доступности).

СТИЛЬ:
- Плоский вектор: сплошные заливки + простые градиенты. Скруглённые формы.
- Без имитации живописи, без шума, без тяжёлых фильтров-блюров.
- Одна ясная спокойная сцена, крупный понятный силуэт, тёплое доброе настроение.
- Палитра — использовать ТОЛЬКО эти цвета:
  небо #BFE3F0/#8FCFE6, белый #FFFFFF, бирюза #7FB4B0, зелень #9FCF6E/#5E9E5A/#A9C5A0,
  золото/песок #F2C94C/#E9CE8E, коричневый #8B5A2B/#A0653A, коралл #EB8A73,
  кожа #F1C9A5/#D9A878, свечение/солнце #FCE39A, тёмно-синий #3D5A80.

ТЕХНИКА:
- Один <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 1200 900"
  preserveAspectRatio="xMidYMid meet">, фон на всю область.
- Только векторные примитивы. Без <image>, <script>, base64, внешних ссылок.
- До ~150 узлов, до ~15 КБ. Добавь короткий <title>.

КАРТОЧКА:
- Номер вопроса: {{question_number}}
- Вопрос: {{question_content}}
- Ответ: {{answer}}
- Стих: {{verse_reference}} — {{verse_text}}

ЗАДАЧА СЦЕНЫ:
Придумай простую конкретную сцену, понятную ребёнку 6–8 лет, отражающую ОТВЕТ.
Приоритет: творение/природа → ребёнок в действии → библейский предмет-символ → библейская сцена (без Божества).
Рекомендация по сцене: {{scene_brief}}
```

Плейсхолдеры `{{...}}` берутся из `catechism.json`:
- `{{question_number}}`, `{{question_content}}`, `{{answer}}` — поля вопроса.
- `{{verse_reference}}` ← `verses[].reference`, `{{verse_text}}` ← `verses[].text` у **первого** связанного стиха через `question_verses` (минимальный `position`). Если у вопроса нет стихов — блок «Стих: …» опустить.
- `{{scene_brief}}` — короткая подсказка сцены из таблицы §7 или ручная.

Готовый SVG сохранить в `public/illustrations/qNNN.svg` (не в поле JSON). Для растра — тот же каталог и поле `illustration` с соответствующим расширением; этот промпт рассчитан на SVG.

## 9. Чек-лист самопроверки агента (перед выдачей)

Проверять вручную / агентом до сохранения файла. `sanitizeSvg` на сборке **не** проверяет пункты ниже (кроме вырезания скриптов).

- [ ] Нет изображения Бога / Христа / Святого Духа (в т.ч. руки из облака, нимба Божества).
- [ ] Вся женская одежда — платье/юбка ниже колена, без брюк; всё скромно закрыто.
- [ ] Нет наготы (в т.ч. Адам/Ева).
- [ ] Нет пугающих элементов (огонь с людьми, тела, кровь, монстры).
- [ ] Нет текста/букв в рисунке (`<text>` и буквы-формы запрещены).
- [ ] Цвета только из палитры; viewBox `0 0 1200 900`; фон залит.
- [ ] Валидный самодостаточный SVG без `<script>`/`<image>`/base64/внешних URL.
- [ ] Есть `<title>`; сложность в пределах бюджета (~150 узлов, ~15 КБ).
- [ ] Файл лежит в `public/illustrations/qNNN.svg`; `illustration` в JSON — путь, не разметка.
